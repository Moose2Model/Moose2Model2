# Moose2Model2
Moose2Model Version 2
This repository will contain the coding for version 2 of Moose2Model. Currently it contains only test coding. See Moose2Model for a tool that you can use now.
## Installation
Install from this repository using Iceberg on Pharo 8.
Install Roassal2 from https://github.com/ObjectProfile/Roassal2 using Monticello as described there. The installation into Pharo 8 should work. Only the Trachel package should be needed. 

At a later stage of the develompment an adapted version of Trachel (Roassal2) should be required, this might be available at: https://github.com/Moose2Model/Roassal2

## Planned features (Unstructured list)
The idea is to implement a tool to draw circuit diagrams for software. Moose2Model is this doing already. But it is currently quite restricted and not intuitive. Software can be very complex, much higher than mechanical or electronical systems. Moose2Model2 should therfore provide tools and features to make this complexity handable (More than what Moose2Model currently has).

-Supports multiple users

-Supports multiple sources of information

-Supports multiple systems

-Supports multiple computer languages

-Supports multiple granularities

-Supports mutliple value type

-Can handle big models

-Understands transport layers (Development, Stage and Produktion)

-Is able to import various Moose Models

-Is able to read and convert old Moose2Model diagrams
