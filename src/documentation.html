General

Analyze software models in SOMIX format in a JavaScript application. You can run the application from your local file system.

What can be done

The application is currently developed. You may open an issue in the Github repository to report a problem or require a function.

Use SAP2Moose to create an mse file with a SOMIX metamodel to use the application. Or load this example file to your local computer and load it into the application.

Implemented Features

  • Load a SOMIX model
  • Force-dragged diagram. Start and stop with a context click in background of pane
  • Scrolling by dragging the pane
  • Dragging elements
  • Zooming with the wheel of the mouse
  • Context click on elements:
    • Get informations on the element
    • Jump to code when adt links are active in Eclipse or another code editor
    • Remove pinning of recently dragged elements
