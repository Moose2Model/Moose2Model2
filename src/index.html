Moose2Model

Analyze software models in SOMIX format in a JavaScript application. You can run the application from your local file system.

You may read this blog https://blogs.sap.com/2022/08/28/somix-suite-for-software-visualization/.

Use SAP2Moose to create an mse file with a SOMIX metamodel to use the application. Or load this example file to your local computer and load it into the application. There is also the option to load a hard coded demo model directly in the application.

You can also analyze a folder in your local file system to view and search directories and files. The SOMIX model created for this purpose is currently only created internally in the application.

The project is available on Github at https://github.com/Moose2Model/Moose2Model2.

Please join the News Group of this project https://groups.google.com/g/somixsuite.
